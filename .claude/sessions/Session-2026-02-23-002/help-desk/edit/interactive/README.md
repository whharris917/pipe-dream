# Interactive Authoring

Interactive authoring replaces freehand editing with template-driven prompts. Currently supports VR documents only.

**Workflow:**

1. [Check out](../checkout.md) the document
2. [Respond to prompts](respond.md) sequentially
3. [Check progress](progress.md) at any time
4. [Amend a response](amend.md) if needed
5. [Preview compiled output](compile.md)
6. [Check in](../checkin.md) when complete

See `QMS-Docs/08-Interactive-Authoring.md` for the full model.
