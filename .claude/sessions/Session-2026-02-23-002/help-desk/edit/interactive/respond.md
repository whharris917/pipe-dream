# Respond to a Prompt

```
qms interact DOC-ID --respond "Your response"
```

Or from a file:

```
qms interact DOC-ID --respond --file /path/to/content.txt
```

Records the response with author and timestamp, then advances to the next prompt.

**View current prompt:** `qms interact DOC-ID` (no flags)
