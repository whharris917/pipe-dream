# Close an Executable Document

```
qms close DOC-ID
```

Transitions from POST_APPROVED to CLOSED. Terminal state -- no further modifications.

**Prerequisites:**

- Document must be POST_APPROVED
- All child documents must be in appropriate states (see `QMS-Docs/03-Workflows.md`)

**Need to fix something after closure?** [Create an Addendum (ADD)](../create/child/add.md).
