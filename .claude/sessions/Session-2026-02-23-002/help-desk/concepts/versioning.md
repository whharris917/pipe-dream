# Version Numbering

Format: **N.X**

| Version | Meaning |
|---------|---------|
| 0.X | Never been approved |
| N.0 (N >= 1) | Currently approved/effective |
| N.X (X > 0) | Under revision of an effective document |

Minor increments on each check-in. Major increments on each approval. Minor resets to 0 on approval.
