# Create a Requirements Traceability Matrix

```
qms create RTM --title "Your title"
```

Creates a DRAFT RTM at v0.1. Maps each RS requirement to its implementation and verification evidence.

Must include a Qualified Baseline section with an actual CI-verified commit hash (not "TBD"). See `QMS-Docs/10-SDLC.md`.
