# Create a Verification Record

```
qms create VR --title "Your title" --parent CR-NNN
```

VRs are pre-approved evidence forms. Born at v1.0 in IN_EXECUTION -- no pre-review phase.

VRs are authored exclusively through [interactive authoring](../../edit/interactive/README.md).

See `QMS-Docs/07-Child-Documents.md` for content requirements.
