# Create a Variance Report

```
qms create VAR --title "Your title" --parent CR-NNN
```

Use when a non-test execution item cannot proceed as planned. Parent must be IN_EXECUTION.

**Type 1:** Parent cannot close until VAR is CLOSED.
**Type 2:** Parent can proceed once VAR is PRE_APPROVED.

See `QMS-Docs/07-Child-Documents.md` for scope handoff requirements.
