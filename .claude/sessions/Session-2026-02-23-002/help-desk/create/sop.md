# Create an SOP

```
qms create SOP --title "Your title"
```

Creates a DRAFT SOP at v0.1. SOPs are non-executable -- they define procedures, not implementation work.

**Lifecycle:** DRAFT > IN_REVIEW > REVIEWED > IN_APPROVAL > EFFECTIVE
