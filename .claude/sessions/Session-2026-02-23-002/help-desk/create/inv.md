# Create an Investigation

```
qms create INV --title "Your title"
```

Creates a DRAFT INV at v0.1. Use when something went wrong and you need root cause analysis.

Requires: deviation description, impact assessment, root cause analysis, and CAPAs. See `QMS-Docs/05-Deviation-Management.md`.
