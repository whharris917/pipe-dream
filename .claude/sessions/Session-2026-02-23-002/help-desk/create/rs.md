# Create a Requirements Specification

```
qms create RS --title "Your title"
```

Creates a DRAFT RS at v0.1. An RS defines what a system shall do -- requirements only, no implementation details.

Requirement IDs follow `REQ-{TYPE}-{NNN}` (e.g., `REQ-MCP-001`). See `QMS-Docs/10-SDLC.md`.
