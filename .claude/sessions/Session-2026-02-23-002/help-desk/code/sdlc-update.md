# Update SDLC Documents

When a CR changes code in a governed system:

1. Update the **RS** if requirements changed ([create](../create/rs.md) or revise)
2. Update the **RTM** with new/changed verification evidence ([create](../create/rtm.md) or revise)
3. Record the qualified commit hash in the RTM's Qualified Baseline section
4. Both must be EFFECTIVE before the CR can close

Even pure refactors need an RTM update with the new qualified commit.
