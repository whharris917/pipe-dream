# Genesis Sandbox

For new systems without requirements or clear scope:

- Branch pattern: `genesis/{system-name}`
- No CRs, no reviews, no SDLC documents
- Purely exploratory

**Exit:** Create an Adoption CR that establishes the first RS, RTM, qualified commit, and merges to `main`.

After adoption, all changes require full change control.

See `QMS-Docs/09-Code-Governance.md`.
