# An Execution Item Failed

Mark the EI outcome as **Fail** and create a child document:

- **Non-test document** (CR, INV): [Create a VAR](../create/child/var.md)
- **Test protocol** (TP): [Create an ER](../create/child/er.md)

The child document captures what went wrong, why, and how it's resolved. Reference it in the EI's evidence column.

You cannot skip or delete the EI. See [Scope Integrity](../concepts/scope-integrity.md).
