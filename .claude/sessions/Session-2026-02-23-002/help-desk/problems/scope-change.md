# Need to Change Approved Scope

If you're still in execution, [create a VAR](../create/child/var.md). The VAR documents:

- What deviated and why
- Impact on parent execution items
- Resolution plan
- Scope handoff (what moves from parent to child)

You cannot silently drop or modify approved EIs. See [Scope Integrity](../concepts/scope-integrity.md).
