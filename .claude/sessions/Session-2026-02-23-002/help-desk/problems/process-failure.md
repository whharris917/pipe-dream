# A Process Wasn't Followed

[Create an Investigation (INV)](../create/inv.md).

Use INVs for systemic issues, unclear root causes, or process failures. The INV requires:

- Root cause analysis
- Corrective and Preventive Actions (CAPAs)
- CAPA implementation via child CRs

See `QMS-Docs/05-Deviation-Management.md` for guidance.
